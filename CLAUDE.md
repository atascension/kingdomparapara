# Kingdom Para Para — Project Requirements

## Stack
- Single `index.html` file — no build tools, no frameworks
- Tailwind CSS 4.2.2 via CDN

## Content
All website copy must follow `website_copy/COPY.md` exactly.

## Design & Styling
All visual design — colours, typography, layout, and imagery direction — must follow `brand_identity/BRAND_IDENTITY.md`.

## Design Skills

When working on any visual design or frontend implementation for this project, Claude must act as the lead designer and invoke the following skills before beginning any design work:

1. **`high-end-visual-design`** — for premium, agency-level design execution
2. **`design-taste-frontend`** — for UI/UX engineering and interface architecture
3. **`redesign-existing-projects`** — when improving or refining an existing implementation

Do NOT use `minimalist-ui` (conflicts with the brand's rich colour palette) or `industrial-brutalist-ui` (wrong aesthetic for this brand).

All design decisions must be grounded in `brand_identity/BRAND_IDENTITY.md` — colours, typography, imagery themes, and tone must be respected exactly as defined.
